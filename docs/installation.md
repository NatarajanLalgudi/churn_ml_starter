# Installation

## Stable release

To install churn_ml_project, run this command in your terminal:

```sh
uv add churn_ml_starter
```

Or if you prefer to use `pip`:

```sh
pip install churn_ml_starter
```

## From source

The source files for churn_ml_project can be downloaded from the [Github repo](https://github.com/NatarajanLalgudi/churn_ml_project).

You can either clone the public repository:

```sh
git clone git://github.com/NatarajanLalgudi/churn_ml_project
```

Or download the [tarball](https://github.com/NatarajanLalgudi/churn_ml_project/tarball/master):

```sh
curl -OJL https://github.com/NatarajanLalgudi/churn_ml_project/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd churn_ml_project
uv pip install .
```
